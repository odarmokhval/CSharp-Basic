1. блоки операторів розділяються крапкобю з комою: ;
2. файл компілятора в C# називається збірка (assembly)
3. в даному коді неправильно розтавлені літерали. { має об'являти закінчення методу.
Має бути так:

static void Main(string[] args)
{
	Console.WriteLine("Hello world") ;
	Console.ReadLine() ;
}
